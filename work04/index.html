動画制作

[ サービスについて ]

素材を支給するだけ！ターゲットの心を掴む動画を制作！

「撮影したものの自分ではうまく編集できない」「会社や商品をうまくアピールできない」「他社で制作できないと断られてしまった」 「納品までもう時間がない」「予算に限りがあり、制作費を安く済ませたい」「どこに依頼していいかわからない」など

自社では動画制作できない・他で断られてしまった方のために撮影した動画や写真、会社案内・商品パンフデータなど支給していただいた素材から動画制作・動画編集するサービスです。

フリーランスとして長年制作に携わる経験豊富なプロの動画クリエイターが、あなたの会社や商品・サービスを力強くアピールする動画を制作します。

[ 動画制作の流れ ]

  • 01

    無料相談

    「制作費はどのくらいになるの？」「急ぎの案件だけど対応できる？」「このサービスはどういう見せ方がいいの？」など、さまざまな「わからない」にお答えします。

    初めて動画制作を依頼する方や過去の動画制作で上手くいかなかった方が安心してご相談できるよう、経験豊富な動画クリエイターが丁寧に対応いたします。

    ご予算に合った最適なプランや目的に合った効果的な見せ方をご提案し、お客様のニーズを最大限に考慮した動画制作を実現いたします。まずはお気軽にお問い合わせください。

  • 02

    ヒアリング

    ヒアリングは動画制作の成否を決める大きな要素です。「こんな仕上がりを希望していなかった…」「イメージがかけ離れてる…」など認識のズレがないように、動画制作の依頼背景や目的、どのような効果を期待するのかなど確認いたします。

    ヒアリングシートを使用しますが、詳細や具体的な内容が決まっていなくても問題ありません。お客様のざっくりとしたイメージや、YouTubeなどからの参考動画をお見せしていただいても問題ありません。

  • 03

    ご提案

    ヒアリングの情報をもとに、動画制作の企画・構成・画コンテをご提案致します。これは動画制作の中で最も重要な工程であり、お客様の動画作成の目的や期待する効果を共通認識として企画・構成を練っていきます。

    動画の種類については、実写かインフォグラフィックの2種類があります。どちらの方向性で動画を進めるのか、お客様に選択していただきますが、迷われる場合にはこちらからご提案させていただきます。

    インフォグラフィック動画

    インフォグラフィック動画は、図やグラフ、アイコンなどをビジュアルとして伝えることができるため、商品やサービスの特徴や利点を効果的に伝えるのに適しています。複雑な情報を整理し、視覚的に魅力的なデザインで表現することで、ターゲットに関心や興味を持ってもらいやすくなります。

    実写編集動画

    実写編集動画は会社紹介 / 商品・サービス紹介 / インタビュー / セミナー / マニュアル / イベントなど生の声や臨場感を求めるシーンに適しており、共感や説得力を生み出し、ブランディングにも効果的です。
    表現方法は多岐に渡り、例えば会社の理念やサービスを語るインタビューシーンにサービス内容を連想させる動画・画像を挿入したり、若手社員の現場の様子にテロップやモーションを加えて話の内容に印象を持たせるなど、より視聴者の共感を誘う動画にすることができます。

  • 04

    ご発注

    構成・お見積りにご納得いただきましたら、正式に発注となります。この段階で映像や動画の著作権や2次使用権、守秘義務契約内容などご契約に関するお手続きをしていただきます。これにより、制作物の権利や契約内容が明確化され、双方が納得した上での正式な発注となります。

    ご契約に関する事項については、事前に十分にご説明し、お客様のご理解をいただいた上で進めさせていただきます。ご契約にはオンライン上で契約締結ができるクラウドサイン（https://www.cloudsign.jp/）を利用しますので、スムーズに制作を開始できます。契約締結後、お見積額の50%を着手金としてお振込いただき、制作開始となります。

    ※ただし10万円以下の制作費の場合、全額ご入金いただきましてからの制作着手となります。

  • 05

    制作開始

    企画に合わせて映像、動画制作をスタートします。ヒアリングや企画・構成の情報をもとに、制作のスケジュールを組み立て、制作に必要なリソースや素材の手配を行います。編集、CGやアニメーションの制作、ナレーションの録音など、制作プロセスの各段階での進捗管理を行いながら、クオリティの高い制作を行います。

    制作の進捗状況や納期については、お客様に定期的に報告し、ご確認いただきながら制作を進めていきます。お客様のご要望に応えるために最善を尽くします。

  • 06

    編集＆映像チェック

    お客様のご要望に合わせて、映像のカット編集やシーンの並び替え、テキストやグラフィックの追加などを行い、最終的な動画を仕上げます。

    初稿の映像は通常7～10営業日程度で提出させていただきますが、お急ぎの場合には短納期も可能です。
    初稿の映像が完成した際には、お客様にご確認いただき、フィードバックをいただき、修正対応いたします。

  • 07

    納品

    映像の修正が完了し、最終的にお客様からご了承いただいた映像をご希望の形式で納品いたします。

    納品形式については、WEB用データ / DVD・ブルーレイなどお客様のご要望に合わせて対応いたします。

[ よくある質問 ]

  • Q.動画に関する知識がありません。それでも依頼できますか？

    はい可能です。プロの動画クリエイターがサポートいたしますのでご安心ください。
    まずはヒアリングシートに記載していただき、ヒアリング内容や資料の精査で的確にニーズを拾い上げ、依頼者が望むゴールを目指し最適な動画をご提案いたします。

  • Q.オンラインの打ち合わせも可能でしょうか？

    はい。打ち合わせは基本的にオンラインのみで行います。ZOOM/Google Meet/Slackなどが利用できます。

  • Q.やりとりはメールのみでしょうか？

    メール以外に、Chatwork などの連絡ツールが利用できます。

  • Q.ヒヤリングシートはいつ送っていただけるのでしょうか？

    まずお問い合わせフォームよりご連絡いただきまして、弊社よりGoogleスプレッドシートにてヒアリングシートを共有させていただきます。 そのスプレッドシートに記載いただきましてから、オンラインでミーティングさせていただきます。

  • Q.地方からの依頼でも大丈夫ですか？

    はい。問題ございません。日本全国からのご依頼に対応可能です。

  • Q.お見積もりは無料ですか？

    無料です。お問い合わせフォームよりご連絡をいただき、ご要望などヒアリングさせていただきます。
    それらの内容をもとに、構成を作成させていただきます。ここまで無料とさせていただいております。ぜひお気軽にお問い合わせください。

  • Q.個人での発注依頼は可能ですか？

    ご依頼可能です。一般個人の方々にも対応させていただいております。

  • Q.途中でキャンセルは出来ますか？

    途中キャンセルも可能です。構成が完了し制作に入っている場合は、キャンセル料として制作費の50％を頂きます。ただし、着手金を頂いている場合、追加でキャンセル料は発生致しません。

  • Q.制作の料金はどれぐらいかかりますか？

    撮影素材・画像・ロゴなどをクライアント様でご準備いただくことが条件となりますが、30,000円から制作が可能です。
    クライアント様で素材を揃えられない場合、無償または有償の素材を使用しますので金額が変動します。また、ナレーションの選択（機械音声か人の音声か）によっても金額が変わります。ぜひお気軽にご相談ください。

  • Q.予算に限りがあります。ご相談に乗っていただけますか？

    はい、ご予算に合わせて、御社の目的に沿った提案をさせていただきます。予算に応じた最適な制作プランを一緒に検討いたしましょう。

  • Q.着手金は必要ですか？

    はい、ご契約後、制作費の50%を着手金としてお振込いただきます。お振込確認後、制作を開始いたします。ただし10万円以下の制作費の場合、全額ご入金いただきましてからの制作着手となります。
    納品後、翌月末までに残額をお振込いただきます。3ヶ月以上の長期制作の場合、中間検収で残額の一部をご請求することがございます。

  • Q.支払はどの段階で発生するのでしょうか？

    ヒアリングシートに記載いただいた内容をもとに、お電話またはZOOM等で打ち合わせを行い、構成を作成します。構成をご確認いただき、ご納得いただけましたら、お支払いしていただきます。 ※10万円以下は、全額前払いとさせていただいております。

  • Q.制作会社の依頼にも対応していただけますか？

    はい、制作会社様からの依頼も承ります。構成や素材（イラスト、画像、撮影素材）が整っていれば、すぐに制作を開始できます。納期が短い案件については、別途料金が発生する可能性がありますが、まずはお気軽にご相談ください。

  • Q.代理店の依頼にも対応していただけますか？

    はい、代理店様からの依頼も承ります。内制で対応できない案件や困難な案件など、お気軽にご相談ください。柔軟に対応させていただきます。

  • Q.CG・3DCGでの動画制作は可能ですか？
